Campione di riferimento

Note introduttive

  • In questa pagina vengono presentate le informazioni relative al campione di riferimento utilizzato per la costruzione delle tavole normative percentiliche relative a: {{ data.metric_config.title }}. Tali tavole consentono la conversione dei punteggi grezzi in punteggi standardizzati.

    L'affidabilità delle tavole normative dipende dalla precisione dei percentili usati come soglie di conversione e la precisione di questi ultimi è funzione della numerosità campionaria. Per percentili non estremi (10°— 90°) un numero di osservazioni nell'ordine delle centinaia garantisce errori standard accettabili; per percentili estremi (1°— 10°, 90°— 99°) sono necessarie migliaia di osservazioni, data la scarsità di valori nelle code della distribuzione.

  • Sempre in questa pagina, viene presentata la procedura di fitting di una famiglia di distribuzioni teoriche {% if data.metric_config.metric_type == "time" %}(normale, log-normale, gamma, weibull){% else %}(binomiale negativa, Poisson, Poisson con inflazione di zeri){% endif %} ai dati campionari.

    Il modello con migliore adattamento (fit) sarà utilizzato per validare la stabilità delle stime percentiliche attraverso il metodo della simulazione Monte Carlo {% if header_numbering %}(cfr allegato {{ header_numbering[1] }}){% else %}(vedi pagina successiva){% endif %}.

    La bontà dell'adattamento viene qui valutata attraverso indici di adeguatezza e analisi grafica mediante {% if data.metric_config.metric_type == "time" %}Q-Q plot{% else %}Rootogram{% endif %} che quantificano la capacità dei modelli di riprodurre le caratteristiche distribuzionali del campione empirico.

Percentili bootstrap e validazione Montecarlo

Note introduttive

  • La tecnica delle repliche bootstrap rappresenta un approccio non parametrico per stimare i percentili campionari senza assumere specifiche distribuzioni teoriche. Il procedimento usato genera {{ data.metric_config.bootstrap_n_replicates }} campioni bootstrap (N={{ "%.0f" | format(data.clean.descriptive_stats.count) }}) attraverso ricampionamento casuale con sostituzione del dataset originale.

    Per ciascuna replica bootstrap vengono calcolati i percentili di interesse, ottenendo una collezione di stime. Il valore finale è rappresentato dalla loro mediana, mentre la variabilità fornisce una misura dell'errore standard.

  • La validazione dei percentili bootstrap tramite simulazione Monte Carlo costituisce un passaggio cruciale per verificare la coerenza tra le stime empiriche ottenute con la tecnica bootstrap e quelle del modello teorico. Il processo inizia identificando la distribuzione probabilistica che meglio descrive i dati osservati. {% if header_numbering %}(cfr allegato {{ header_numbering[0] }}){% else %}(vedi pagina precedente){% endif %}.

    Una volta selezionato il modello ottimale e stimati i parametri, si procede alla generazione di campioni teorici mediante simulazione Monte Carlo, ciascuno della stessa dimensione dell'originale. Per ogni campione simulato vengono calcolati gli stessi percentili del procedimento bootstrap, creando una distribuzione teorica di riferimento.

    Il confronto tra percentili bootstrap e teorici avviene attraverso analisi statistiche che valutano differenze assolute e relative, e rappresentazioni grafiche. Una buona concordanza indica che le stime bootstrap sono affidabili.
